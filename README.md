# 💰 Paisa Manager Pro

Personal Finance Tracker with Login + Guest Mode

## Features
- 🔐 Login / Register (data Supabase cloud pe save)
- 👤 Guest Mode (data phone pe local save)
- 📅 Monthly tracking with calendar view
- 📊 Category breakdown with budget limits
- 🔁 Recurring expenses (auto monthly)
- 📈 12-month history chart
- ⬇️ CSV & JSON export
- 📱 PWA — phone pe install ho sakti hai

## GitHub Pages Setup

1. Yeh repo GitHub pe upload karo
2. Settings → Pages → Source: `main` branch → `/ (root)`
3. Save karo — 2 min mein live ho jayega
4. URL milega: `https://yourusername.github.io/paisa-manager/`
5. Phone mein woh URL kholo → Share → "Add to Home Screen"

## Files
- `index.html` — Main app
- `manifest.json` — PWA config
- `sw.js` — Service worker (offline support)
- `icons/` — App icons (192x192 aur 512x512 PNG)
